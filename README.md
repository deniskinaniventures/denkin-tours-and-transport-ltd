# DenKin Tours and Transport Website

This is the private development repository for the official DenKin Tours and Transport website.

## Project Overview
The website showcases our tour services, vehicles, itineraries, and contact information. It is designed for both desktop and mobile users.

## Folder Structure
- `index.html` – main homepage
- `css/` – stylesheets
- `js/` – JavaScript files
- `images/` – all images used on the site

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/yourusername/denkin-tours-website.git
2. Open `index.html` in your browser.
